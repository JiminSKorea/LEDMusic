# Real-Time LED MUSIC VISUALIZER: THE CHIC 909

## How to Set Up
1. Connect the product to the Laptop with the USB cable
2. Open maincode.ino with Arduino IDE
3. Set the Board as Arduino Uno and the Port of your environment
4. Upload the code to the product.

## How to Use
1. Press the Power button to turn ON/OFF the product
2. Press the Mode Switch button to switch between two different LED visualization modes

## How to Interact
1. Play the music with speakers and place the product near the speakers
2. Enjoy the visualization in different modes
* Rhythmic Mode: Flashes LEDs to the beat of the music
* Flow Mode: Move LED lights from left to right and back again
